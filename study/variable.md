# 변수

## 실제로 사용하는 애들

- 정수 int, long
- 실수 double
- boolean
- 문자열 String, 한 문자라도 String

- 정수 끼리 나누면 해당 값이 소수점이 발생해도 정수(버림)로 처리
- 한 쪽에 실수가 있어야 소수점이 제대로 나온다

```java
int a = 10;
double b = 10.0;
int c = 3;

// a / c = 3
// b / c = 3.333333
```

## 변수 명명 규칙(컴파일 에러 발생)

## 명명 관례

- 변수
    - 소문자로 시작, camelCase 사용
- 클래스
    - 대문자로 시작, PascalCase 사용
- 상수
    - 전부 대문자 사용, 언더바로 연결 (USER_LIMIT)
- 패키지
    - 전부 소문자 사용, 마침표(.)로 연결 (org.spring.boot)
